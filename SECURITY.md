# Security Policy

This repository is a public-safe demo. Do not commit:

- real RTSP URLs,
- credentials,
- tokens,
- private keys,
- `.env` files,
- private datasets,
- production logs,
- internal deployment configurations,
- model weights that you are not licensed to publish.

If you find sensitive information in this repository, remove it immediately, rotate any affected credentials, and open a private security issue or contact the maintainer directly.

Use placeholder URLs only:

```text
rtsp://username:password@camera-ip:554/stream
```
